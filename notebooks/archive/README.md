# notebooks/archive — Versiones de desarrollo del EDA

Estos notebooks son versiones de trabajo del Análisis Exploratorio de Datos (EDA)
utilizadas durante el desarrollo del proyecto. **No son el entregable canónico.**

El notebook oficial del proyecto es:

```
notebooks/UrbanFlow_CDMX_Colab.ipynb
```

Ese notebook está diseñado para correr en Google Colab sin credenciales externas,
con los datos cacheados inline, y reproduce los resultados de validación reportados
en el artículo técnico (MAPE P50 ≈ 11.9%, cobertura banda P10–P90 ≈ 93.3%).
